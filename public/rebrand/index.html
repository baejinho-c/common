랜딩 페이지 리브랜딩과 AI 진단에 대해 궁금한 점을 정리했습니다.

Rebrand AI는 어떤 서비스인가요?
웹사이트 URL을 입력하면 AI가 브랜드 정체성, 레이아웃, 타이포그래피, 컬러, UX를 진단하고 실시간 HTML 미리보기로 리브랜딩 결과를 확인할 수 있습니다.
무료로 사용할 수 있나요?
API 키 없이 하루 3회까지 무료 진단이 가능합니다. 채팅 수정이나 추가 분석이 필요하면 OpenAI, Anthropic, Gemini API 키를 입력해 이용할 수 있습니다.
어떤 페이지에 적용할 수 있나요?
랜딩 페이지, SaaS 홈페이지, 제품 소개 페이지 등 공개 URL이 있는 웹 페이지를 분석할 수 있으며, HTML 파일 업로드도 지원합니다.
SEO 진단도 함께 받을 수 있나요?
Rebrand AI는 브랜드·디자인 중심 진단입니다. 메타 태그·헤딩·링크 등 기술 SEO는 seo-copilot.restyart.com의 SEO Copilot에서 별도로 점검할 수 있습니다.
