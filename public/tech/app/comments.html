댓글 시스템

게시글의 댓글을 관리하고 상호작용하세요.

테스트용 게시글 선택
댓글 시스템 기능

지원하는 기능:

  • 댓글 작성: POST /posts/:id/comments - 새 댓글 작성
  • 답글 작성: parent_id를 포함하여 대댓글 작성
  • 댓글 수정: PUT /comments/:id - 댓글 내용 수정
  • 댓글 삭제: DELETE /comments/:id - 댓글 삭제
  • 댓글 목록: GET /posts/:id/comments - 페이지네이션 지원

API 사용법:

POST /posts/1/comments
{ "content": "댓글 내용", "parent_id": 2 }
PUT /comments/1
{ "content": "수정된 댓글 내용" }
DELETE /comments/1
